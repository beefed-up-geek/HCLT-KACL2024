1. 2019회계연도 전략물자 수출입 통제 기반 구축 사업의 결산 현황에 따르면 전략물자 수출입 통제 기반 구축 사업의 예산현액은 68억 6,600만 원이며 전액이 집행되었다.

3. 68억 6,600만 원이라는 금액이 전략물자 수출입 통제 기반 구축 사업의 집행액과 예산현액이었는데, 이는 2019회계연도 전략물자 수출입 통제 기반 구축 사업의 결산과 관련한 금액이다.

4. 전략물자 수출입 통제 기반 구축 사업의 예산현액은 68억 6,600만 원이며 전액이 집행되었다는 결과는 2019회계연도 전략물자 수출입 통제 기반 구축 사업의 결산 현황에 나타나 있다.

5. 2019회계연도 기준, 전액이 집행된 68억 6,600만 원의 전략물자 수출입 통제 기반 구축 사업의 예산현액은 전략물자 수출입 통제 기반 구축 사업의 결산 현황에서 나타났다.
