정답들

1. 지리적 표시 단체표장과 지리적 표시 등록제를 비교해보면, 지리적 표시 단체표장은 2004년에 도입한 상표법을, 지리적 표시 등록제는 1999년에 도입한 농산물품질관리법을 근거법률로 한다.

2. 지리적 표시 단체표장과 지리적 표시 등록제의 비교 상황에는, 지리적 표시 단체표장과 지리적 표시 등록제는 각각 2004년에 받아들인 상표법과 1999년에 받아들인 농산물품질관리법을 근거법률로 드러냈다.

3. 지리적 표시 단체표장과 지리적 표시 등록제의 근거법률을 비교해보면, 2004년에 도입한 상표법과 1999년에 도입한 농산물품질관리법은 각각 지리적 표시 단체표장과 지리적 표시 등록제의 근거법률이 된다.

4. 지리적 표시 단체표장과 지리적 표시 등록제를 비교하면, 지리적 표시 단체표장은 2004년 도입한 상표법을 법적근거로 삼으며 지리적 표시 등록제는 1999년 도입한 농산물품질관리법을 법적근거로 삼는다.

5. 지리적 표시 단체표장과 지리적 표시 등록제를 비교해 봤을 때, 2004년에 도입한 상표법을 근거법률로 두는 것은 지리적 표시 단체표장이고, 1999년에 도입한 농산물품질관리법을 근거법률로 두는 것은 지리적 표시 등록제이다.
